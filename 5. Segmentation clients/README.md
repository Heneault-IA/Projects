# 🛒 Projet 5 : Segmentation Client E-commerce (Olist Brésil)

## 🎯 Objectif du Projet

Ce projet se concentre sur les données de la marketplace brésilienne **Olist**. La mission est de réaliser une segmentation des clients en utilisant des techniques d'apprentissage non supervisé, afin d'aider les équipes marketing. Le projet inclut également une simulation pour évaluer à quelle fréquence le modèle de segmentation doit être mis à jour (maintenance au cours du temps).

## 🗂️ Données Nécessaires

Le dataset provient de Kaggle :
🔗 [Dataset Olist Brazilian E-commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

⚠️ **Instruction** : Téléchargez les données et placez-les dans le dossier `Data`.

## 🛠️ Outils et Environnement

Environnement préconisé : `python >=3.9, <3.12`

- **Traitement et Visualisation** : `pandas` (^2.1.0), `numpy` (^1.25.2), `matplotlib` (^3.7.2), `seaborn` (^0.12.2), `yellowbrick` (^1.5)
- **Machine Learning** : `scikit-learn` (^1.3.0), `xgboost` (^2.0.0), `lightgbm` (^4.1.0)
- **Interprétabilité et Modélisation avancée** : `shap` (^0.42.1), `lime` (^0.2.0.1), `statsmodels` (^0.14.0), `hyperopt` (^0.2.7)
