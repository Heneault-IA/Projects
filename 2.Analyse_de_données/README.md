# 📊 Projet 2 : Analyse de Données - Expansion Stratégique

## 🎯 Objectif du Projet

Ce projet utilise les données de la **Banque Mondiale (World Data Bank)** concernant l'éducation. L'objectif principal est de réaliser une étude préalable afin d'évaluer la possibilité d'expansion d'une école de formation en ligne à l'international.

## 🗂️ Données Nécessaires

Les données utilisées proviennent du catalogue de la Banque Mondiale :
🔗 [World Bank Data Catalog](https://datacatalog.worldbank.org/search/dataset/0038480)

⚠️ **Instruction** : Les fichiers CSV téléchargés doivent être placés dans le dossier `Data` à la racine de ce répertoire.
