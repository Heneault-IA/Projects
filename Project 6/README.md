# 🖼️ Projet 6 : Classification d'Images et Traitement du Langage Naturel

## 🎯 Objectif du Projet

L'objectif est d'étudier la faisabilité d'un moteur de classification automatique basé sur des descriptions textuelles et des images de produits pour une place de marché. Ce projet aborde des techniques avancées de Deep Learning tant pour le NLP (Natural Language Processing) que pour la Computer Vision (CV).

## 🗂️ Données Nécessaires

🔗 [Télécharger les données images depuis Google Drive](https://drive.google.com/drive/folders/1RJQPqPf-XqB-nLYNcNyLLqdxZe58dHc4?usp=sharing)

⚠️ **Instruction** : Les fichiers téléchargés doivent être extraits dans le dossier `Data/Images`.

## 🛠️ Outils et Environnement

Environnement préconisé : `python >=3.11, <3.12` (l'environnement de développement était sur Windows, d'où la version très spécifique de Tensorflow).

### Librairies principales :

- **Deep Learning & NLP** : `tensorflow` (2.15.0), `torch` (^2.1.1), `transformers` (^4.36.1), `nltk` (^3.8.1), `gensim` (^4.3.2)
- **Traitement d'Image** : `opencv-python` (^4.8.1.78), `pillow` (^10.1.0)
- **Data & Visualisation** : `pandas` (^2.1.3), `scikit-learn` (^1.3.2), `wordcloud` (^1.9.2), `plot-keras-history` (^1.1.38)
