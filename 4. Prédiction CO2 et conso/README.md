# 🏙️ Projet 4 : Prédiction de Consommation d'Énergie et d'Émissions CO2 (Seattle)

## 🎯 Objectif du Projet

À partir des données de la ville de Seattle (2016), le but de ce projet est de réaliser une analyse exploratoire puis de concevoir des modèles de Machine Learning capables de prédire simultanément **deux cibles principales** pour les bâtiments non destinés à l'habitation :

1. Les émissions de CO2.
2. La consommation totale d'énergie.

## 🗂️ Données Nécessaires

Les données nécessaires sont **déjà incluses** dans le répertoire du projet. Aucune action supplémentaire n'est requise.

## 🛠️ Outils et Environnement

Environnement préconisé : `python >=3.9, <3.12`

### Librairies principales :

- **Traitement de données** : `numpy` (^1.25.2), `pandas` (^2.1.0)
- **Visualisation** : `matplotlib` (^3.7.2), `seaborn` (^0.12.2)
- **Machine Learning** : `scikit-learn` (^1.3.0), `xgboost` (^2.0.0), `lightgbm` (^4.1.0)
- **Interprétabilité des modèles** : `shap` (^0.42.1), `lime` (^0.2.0.1)
- **Analyses statistiques et Optimisation** : `statsmodels` (^0.14.0), `hyperopt` (^0.2.7)
